# 🏟️ Cancha Libre - Web Client

Interfaz de usuario moderna para la plataforma **Cancha Libre**. Este cliente permite a los usuarios seguir el fútbol profesional y gestionar sus propios "picados" con una experiencia de usuario fluida y reactiva.

## ✨ Funcionalidades Key

* **Dashboard de Seguimiento:** Visualización de partidos, resultados y estadísticas en tiempo real.
* **Generador de Equipos Pro:** Interfaz interactiva para cargar jugadores y generar equipos nivelados al instante.
* **Gestión de Perfiles:** Visualización del progreso y nivel de habilidad del usuario y sus amigos.
* **Diseño Mobile-First:** Preparado para una transición fluida hacia una aplicación móvil nativa.

## 🛠️ Stack Tecnológico

* **Librería:** React.js
* **Estilos:** Tailwind CSS / CSS Modules
* **Estado:** Context API / Redux Toolkit
* **Consumo de API:** Axios / Fetch

## 📦 Instalación y Uso

1. Clonar el repositorio: `git clone https://github.com/tu-usuario/cancha-libre-web.git`
2. Instalar dependencias: `npm install`
3. Iniciar entorno de desarrollo: `npm run dev`
